Pontszámító kalkulátor

Ez az alkalmazás egy webes pontszámító kalkulátor lesz felvételihez, mely saját adatok, tanulmányi, egészségügyi illetve szociális szempontok alapján adja a pontszámot. Célja, hogy átláthatóbb és egyszerűbb legyen a jelentkezési procedúra. Az ELTE jelenlegi rendszerében élő hallgatóknak nyújtana segítséget. A projektet Java alapú technológiákkal tervezzük megvalósítani.
